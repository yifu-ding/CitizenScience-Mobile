# CitizenScience Mobile

CitizenScience Mobile is a Flutter-based mobile application developed as a team project for the Software Engineering course at Beihang University.

The project aims to build a citizen science platform where users can contribute public scientific information through a mobile interface. Users can upload observations, records, images, and related descriptions, supporting community-driven data collection and information sharing.

## Project Background

Citizen science encourages the public to participate in scientific data collection and knowledge sharing. This project explores how a mobile application can support such participation by providing a simple and accessible interface for submitting and managing scientific observations.

As a software engineering course project, this repository focuses not only on application functionality, but also on software development practices, including requirement analysis, modular design, team collaboration, version control, testing, and documentation.

## Features

* User registration and login
* Submission of scientific observations and records
* Image and text-based information upload
* Browsing and managing submitted records
* Mobile-friendly user interface
* Cross-platform development with Flutter
* Modular project structure for team-based development

## Tech Stack

* Flutter
* Dart
* Mobile application development
* Client-server interaction
* Software engineering project management

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yifu-ding/CitizenScience-Mobile.git
cd CitizenScience-Mobile
```

Install dependencies:

```bash
flutter pub get
```

Run the app:

```bash
flutter run
```
